# heito.xyz (Mango version) Nuxt

## Setup

Make sure to install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3002`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

## ENV File

Create ` .env ` file.

```ts
API_URL=<string>
```
